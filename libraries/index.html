Pustaka

Menggunakan pustaka/library

Library p5.js dapat berupa kode JavaScript apapun yang dapat menambah fungsionalitas inti. Ada dua kategori library. Library inti (p5.dom dan p5.sound ) adalah bagian dari distribusi p5.js, sementara library yang berkontribusi dikembangkan, dimiliki, dan dikelola oleh anggota komunitas.

Untuk memasukkan library kedalam sketsa, tautkan ke file HTML (dibawah tautan file p5.js). Contohnya seperti ini:

<!doctype html<
<html>
<head>
  <script src="p5.js"></script>
  <script src="p5.sound.js"></script>
  <script src="sketch.js"></script>
</head>
<body>
</body>
</html>

Buat library/pustaka-mu sendiri

p5.js menyambut library yang disumbangkan oleh orang lain! Cek tutorial pustaka/library untuk lebih spesifik tetang cara membuatnya. Jika Anda telah membuat library dan ingin menyertakannya dihalaman ini, kirimkan formulir ini!

Memulai

Halaman ini menuntun kepada pengaturan proyek p5.js dan membuat sketsa pertamamu. Jika Anda ingin memulai dengan p5.js Editor, Anda dapat menuju ke Sketsa Pertama Anda.

Unduh dan Pengaturan File

Cara termudah untuk memulai adalah dengan menggunakan contoh kosong yang disertakan dengan unduhan lengkap p5.js .

Jika Anda melihat isi index.html, Anda akan melihat sumber/tautannya menuju file p5.js. Jika Anda ingin menggunakan versi yang 'diperkecil' (dikompres untuk pemuatan halaman yang lebih cepat), ubah sumber/tautan file-nya menjadi p5.min.js.

<script src="../p5.min.js"></script>

Atau, Anda dapat menautkan ke file p5.js yang dihosting online. Semua versi p5.js disimpan dalam CDN (Content Delivery Network). Anda dapat melihat riwayat versi ini di sini: p5.js CDN. Dalam hal ini Anda dapat mengubah sumber/tautan menjadi: 

<script src="https://cdn.jsdelivr.net/npm/p5@[p5_version]/lib/p5.js"></script>

Contoh halaman HTML akan terlihat seperti ini:

<html>
  <head>
    <script src="https://cdn.jsdelivr.net/npm/p5@[p5_version]/lib/p5.js"></script>
    <script src="sketch.js"></script>
  </head>
  <body>
  </body>
</html>

Anda juga dapat memulai dengan template ini dari codepen.

Lingkungan Editor

Anda dapat menggunakan code editor pilihan Anda. Petunjuk untuk menyiapkan dengan Sublime Text 2 disertakan dibawah ini, opsi editor lainnya bisa menggunakan Visual Studio Code, Brackets, Atom dan OpenProcessing. Notepad++ Eclipse.

Buka Sublime. Ke menu File dan pilih Open... dan pilih lokasi folder file HTML dan JS Anda. Pada sidebar sebelah kiri, Anda akan melihat nama folder-nya diatas, dengan daftar-daftar file yang berada dialam folder tersebut.

Klik file sketch.js dan itu akan terbuka disebelah kanan dimana Anda dapat mengeditnya.

Buka file index.html di web browser Anda dengan mengklik dua kali (double klik) pada file manager/windows explorer atau ketik: file:///lokasi/file/html pada address bar web browser untuk melihat sketsa.

Sketsa Pertama Anda

Pada bagian ini, jika Anda pengguna Processing mungkin akan mengecek ke tutorial Processing transition.

Code snippet with ellipse

Di editor, ketik:


function setup() {

}

function draw() {
  ellipse(50, 50, 80, 80);
}

Maksud dari baris-baris kode diatas adalah "menggambar elips, dengan titik tengah diambil 50 piksel dari kiri dan 50 piksel kebawah dari atas, dengan lebar dan tinggi 80 piksel".

Save/Simpan file sketch dan refresh web browser untuk melihat perubahannya. Jika Anda telah mengetik semuanya dengan benar, Anda akan melihat ini muncul dijendela web browser:

Note for screenreader users

Jika Anda menggunakan screen reader (karena keterbatasan), Anda harus mengaktifkan output yang dapat diakses pada editor online p5, diluar editor Anda harus menambahkan pustakan/library aksesibilitas pada HTML Anda. Pelajari lebih lanjut tentang menggunakan p5 dengan screen reader dan pustakan/library aksesibilitas.

Jika Anda salah mengetik, mungkin tidak akan melihat apa-apa/kosong. Jika ini terjadi, pastikan telah menyalin contoh kode dengan tepat: angka-angka harus dimasukkan dalam tanda kurung dan memiliki koma di antara masing-masing, dan setiap baris harus diakhiri dengan tanda titik koma.

Salah satu hal paling sulit tentang memulai pemrograman adalah Anda harus sangat spesifik tentang sintaksis. Web Browser tidak cukup pintar untuk mengetahui apa yang dimaksud, dan bisa sangat cerewet tentang penempatan tanda baca. Anda akan terbiasa dengan sedikit latihan. Bergantung pada Web Browser yang digunakan, Anda dapat melihat kesalahan/errors di JavaScript "console". Di Google Chrome, ke menu View > Developer > JavaScript Console.

Code snippet with interaction

Selanjutnya kita akan melompat untuk membuat sketch/sketsa yang lebih menarik. Hapus sintaks dari contoh terakhir, dan silakan coba ini:


function setup() {
  createCanvas(640, 480);
}

function draw() {
  if (mouseIsPressed) {
    fill(0);
  } else {
    fill(255);
  }
  ellipse(mouseX, mouseY, 80, 80);
}

Ini akan mebuat sebuah kanvas dengan lebar 640 dan tinggi 480 piksel, dan mulai menggambar lingkaran putih pada posisi mouse saat ini. Saat tombol mouse ditekan, warna lingkaran berubah menjadi hitam. Ini akan dijelaskan lebih lanjut tentang unsur-unsur program secara rinci nanti. Untuk saat ini, jalankan kodenya (save dan refresh browser), gerakkan mouse, dan klik untuk menghubah warnanya.
